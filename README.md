# CV Portfolio

A responsive React + Vite CV portfolio with a dark glowing gradient design and a simple, human-friendly layout.

## Setup

1. Open a terminal in the `CVProject` folder.
2. Run `npm install`.
3. Run `npm run dev`.

## Customize

- Update `src/App.jsx` with your resume content.
- Adjust colors in `src/styles.css` for a personal branding palette.
- Replace contact details and skill set with your own.
